# TJ-ChaseVoice
猫和老鼠手游语音数据集

## 说明
通过解包获得的猫和老鼠手游的语音数据集（无背景音，非常干净），同一角色的所有语音放置在同一文件夹（除了泰菲），由于可能存在不同的配音，请自行选择

注：泰菲_2文件夹中为除了普通泰菲声线和恶菲之外的声线（包括天菲：星月传奇、月神，剑菲等）

注2：数据集内可能存在没有实际意义的音频，比如语气词，笑声等，请自行选择和排查

注3：命名为add的文件是从其它途径获取的音频，使用UVR5处理过，但仍有可能含杂音
